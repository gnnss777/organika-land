# ORGANIKA Design System

## Based on: Creative Design Skill

### Cores
```css
--primary: #2E5E3A      /* Verde principal */
--primary-light: #4AFF7A /* Verde neon */
--background: #F0EDE6    /* Off-white quente */
--surface: #141410        /* Dark background */
--muted: #8A8A7A
--white: #F0EDE6
```

### Tipografia
- **Display/Titulos**: Playfair Display 900 italic
- **Labels/Grande**: Bebas Neue
- **Corpo**: Inter 300
- **Tecnico**: Space Mono

### Spacing Scale
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px

### Regras de Design
1. Usar semantic tokens ( cores definidas acima )
2. Manter hierarquia visual clara
3. Estados de interacao explicitos
4. Espaçamento consistente
5. Contraste adequado
6. Animações suaves

### Componentes
- Buttons: borda 1px, padding 12px 24px, tracking largo
- Cards: bg-surface, border subtle, p-medium
- Inputs: border, focus ring
